    How this tutorial works

    You already play acoustic guitar, so this course skips basics and focuses on jazz fluency: interval thinking, functional harmony, and voice leading.

    • Learn chord formulas, then build any chord in any key.
    • Practice the same harmony in low, mid, and high neck zones.
    • Train progressions so your comping sounds musical, not random.

    Chord Diagram Schematic

    Red circle = root (1). Black circles = other chord tones (labels inside). `x` = mute, `o` = open, and right-side numbers are left-hand fingers (1=index, 2=middle, 3=ring, 4=pinky).

    Build Chords by Formula

    Pick a root and chord quality. Learn why each chord works from intervals.

    Practice task: Build the same quality from 3 different roots before moving on.

    Map Voicings Up and Down the Neck

    Use one chord quality and move between neck zones without losing groove.

    Practice task: Start in low zone, then shift to mid, then high with the same chord quality.

    Comp Through Common Jazz Progressions

    Generate progressions and focus on smooth voice leading over speed.
